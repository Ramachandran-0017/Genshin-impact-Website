# Genshin-impact-Website
# 🎮 Genshin Impact Website Clone

A responsive and visually engaging website inspired by the world of **Teyvat** from Genshin Impact. This project focuses on modern UI design, structured layouts, and responsive web development using Bootstrap.

---

🚀 Live Demo

🔗 https://ramachandran-0017.github.io/Genshin-impact-Website/

---

📌 Features

* 🌐 Fully responsive design using Bootstrap
* 🎨 Clean and modern UI
* 🧭 Multi-page navigation (Home, Nations, Archons, Contact, Login)
* 🖼️ Visually appealing sections with rich images
* 📱 Mobile-friendly layout

---

🛠️ Tech Stack

* HTML
* CSS
* Bootstrap
